# Jest GOT formData

API integration test with [Jest](https://jestjs.io/) and [GOT](https://github.com/sindresorhus/got) to make file upload request.

## Install and Run

```sh
yarn && yarn test
```

* You need configure `.env` variables first to make it work in your API.
